# Vanity Roles

## Automatic Role Reward for Vanity Advertisers

### Overview

Vanity Roles automatically assigns a configured role to any member who includes the server's vanity invite code in their Discord custom status. When they remove it, the role is removed. This rewards members who advertise the server in their profile.

### Setup

1. Run `/settings vanityroles` to open the Vanity Roles settings.
2. Click **Set Role** and select the role to assign to vanity advertisers.
3. Click **Set Custom Code** and enter your server's vanity invite code (just the code, e.g. `aether` — not the full URL).
4. To disable, click **Clear** to remove the configuration.

### Usage

- The bot monitors presence updates for all members.
- When a member's Discord custom status contains the configured vanity code, the role is assigned automatically.
- When they remove the code from their status (or clear their status), the role is removed automatically.
- Members who already have the matching status when the bot restarts will receive the role on their next presence update.

### Commands

#### /settings vanityroles
- **Description:** Opens the Vanity Roles settings menu to configure the reward role and vanity code.
- **Parameters:** None.
- **Permissions:** Administrator.
- **Available On:** Both.

---

### Tier Requirement

Vanity Roles is available on all tiers with no role or configuration limits.
